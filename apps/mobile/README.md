# Mobile App

Expo React Native app with file-based tab navigation.

## Development

```bash
pnpm dev        # Start Metro bundler
pnpm ios        # Open iOS simulator
pnpm android    # Open Android emulator
```

## Configuration

- **app.json** — Expo plugins (router, splash screen, fonts)
- **tailwind.config.js** — NativeWind with shared UI package paths
- **metro.config.js** — Metro bundler for monorepo support
